Projects

Up until this point, this has been the biggest project I have ever worked on. This sounds crazy but a team of friends and I brought a microwave to hackathon and we decided to hack it. We recreated it as a smart microwave that is Alexa controlled, with a locking mechanism using arduino, with surge prcing and image recognition using google vision and authorize.net. To say the least, there was a lot going on in this project. For this project, I primarily worked on remodeling the microwave and reverse engineering the logic on the controller safely. On the first day of hacking my team and I focused on dismantling the microwave and figuring out how to access the buttons. It was difficult for a while until one of my partners realized it was a capacitive touch pad so it works by setting up certain combinations. After a long and tedious guessing, we were able to find the button combination for "add 30 seconds". This was enough for us to do use a switch to click a button that would add 30 seconds.

From there we focused on the programming aspect. I began working on a python script that would check the outputs of a google vision function my friend was running and that would allow us to pull a value from a dictionary and check what is the time we can use. Unfortunately, we didn't have a database filled with information that we could use to cross check so at the competition we had a limited image selection of popcorn and muffins. I also helped a bit with the arduino to make it so it would rotate a knife so it would keep the microwave locked while it was cooking. The idea of this was to make it so it could be used in a communal setting so anyone could use it but not everyone could steal your food. My partners worked on the server side of things with the Alexa and Authorize.net but it all came together somewhat nicely. We kept everything running using heroku and Google Cloud Platform since we had a developer account in heroku and we were given free GCP credit at the event.

The programming was difficult but ultimately this was one of the coolest, most memorable, and most dangerous projects I have ever made. I learned a lot and this project has inspired me to learn so many new things. Even though, I haven't worked with authorize.net I want to begin working on projects that utilize it in new and unique ways. I also have made it a committment to learn javascript and apply it's abilities in web app development. Overall, this was one of my favorite projects of all time. We also won best IOT/hardware hack and best use of a qualcomm device at the competition.
